7 ¿Cuál es la anatomía de una regla CSS?

rt:Cada una de las reglas (aparte del selector) deben estar encapsulada entre corchetes ( {} ). Dentro de cada declaración, debes usar los dos puntos ( : ) para separar la propiedad de su valor. Dentro de cada regla, debes usar el punto y coma ( ; ) para separar una declaración de la siguiente.

10 ¿A qué se llama DOM y cual es su función?

rt: El modelo de objeto de documento (DOM) es una interfaz de programación para los documentos HTML y XML. Facilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido.
